## 2. 行程執行頁 (Itinerary Screen - Tab 1)

**重點**：高對比度、明確的「預計 vs 實際」狀態、累積距離顯示。

|  SummitMate                              ⚙️       |
+--------------------------------------------------+
|   [ D0 交通 ]   [ D1 上山 ]   [ D2 下山 ]        |

|                                                  |
|  07:00 (預計)           累計 0.0 km              |
|    |                                             |
|    ●  向陽登山口                                  |
|    |  H: 2300m                                   |
|    |  [ ✔️ 實際: 07:05 ]                         | <-- 已打卡
|    |                                             |
+--------------------------------------------------+
|  08:30 (預計)           累計 4.3 km              |
|    |                                             |
|   ◉  向陽山屋 (大休)                             | <-- 目前目標

|    |  [ 📍 點擊查看詳情 ]                        |

## 8. 糧食計畫 (Meal Planner)

**入口**：裝備頁 (Tab 3) > 糧食計畫卡片。
**功能**：Tab 切換天數，管理早/午/晚/行動糧，計算熱量與重量。
**參考資訊**：AppBar 右上角 `i` 按鈕可查看乾燥飯比較圖。

```txt
+--------------------------------------------------+
|  糧食計畫                                  ℹ️    |
+--------------------------------------------------+
|  [ D0 ]   [ D1 ]   [ D2 ]                        |
+--------------------------------------------------+
|  +--------------------------------------------+  |
|  | 總重量 850g             總熱量 2300kcal    |  |
|  +--------------------------------------------+  |
|                                                  |
|  v 🌞 早餐 (1項 • 100g)                   (+)    |
|    - 尾西乾燥飯                    100g / 350kc  |
|                                                  |
|  > 🍚 午餐 (未規劃)                       (+)    |
|                                                  |
|  v 🌙 晚餐 (2項 • 200g)                   (+)    |
|    - 佐竹乾燥飯                    100g / 360kc  |
|    - 乾燥蔬菜包                    100g / 50kc   |
|                                                  |
|  v 🍫 行動糧 (3項 • 150g)                 (+)    |
|    - 鹽糖                          50g  / 200kc  |
+--------------------------------------------------+
```
