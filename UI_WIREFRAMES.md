# UI Visual Wireframes (ASCII Design)

## 1. 啟動與設定頁 (Onboarding Screen)

**情境**：使用者首次開啟 App，尚未設定暱稱時強制顯示。

```txt
+--------------------------------------------------+
|                                                  |
|           歡迎使用 SummitMate                    |
|                                                  |
|      [Logo/山脈圖示]                             |
|                                                  |
|   為了方便隊友辨識，請輸入你的暱稱：             |
|                                                  |
|   +------------------------------------------+   |
|   |  Alex                                    |   |
|   +------------------------------------------+   |
|                                                  |
|           [  開始使用  ]                         |
|                                                  |
+--------------------------------------------------+
```

---

## 2. 行程執行頁 (Itinerary Screen - Tab 1)

**情境**：登山進行中，查看時間軸並進行打卡。
**重點**：高對比度、明確的「預計 vs 實際」狀態。

```txt
+--------------------------------------------------+
|  < D0 交通   |   D1 上山   |   D2 下山 >         | <-- (AppBar Bottom)
+--------------------------------------------------+
|  進度: 向陽山屋 -> 嘉明湖 (目前完成 40%)           |
+--------------------------------------------------+
|  07:00 (預計)                                    |
|    |                                             |
|    O  向陽登山口 [圖片icon]                       |
|    |  H: 2300m | 0K                              |
|    |  [ ✔️ 實際: 07:05 ]                         | <-- (狀態：已完成)
|    |                                             |
+--------------------------------------------------+
|  08:30 (預計)                                    |
|    |                                             |
|   (O) 向陽山屋 (大休)                             | <-- (狀態：目前目標)
|    |  H: 2850m | 4.3K                            |
|    |  [ 📍 按此打卡 ]                            | <-- (點擊彈出 ActionSheet)
|    |                                             |
+--------------------------------------------------+
|  12:00 (預計)                                    |
|    |                                             |
|    O  黑水塘                                     |
|    |  H: ...                                     |
|    |  [ 等待中... ]                              |
|    |                                             |
+--------------------------------------------------+
| [ 行程 ]      [ 📋 共編 ]      [ ⚙️ 工具 ]      | <-- (Bottom Nav)
+--------------------------------------------------+
```

## 3. 協作看板頁 (Collaboration Board - Tab 2)

**情境**：行前討論裝備或行程。
**重點**：巢狀留言顯示、同步狀態提示。

```txt
+--------------------------------------------------+
|  協作看板 (裝備區)              🔄 (10m ago)     |
+--------------------------------------------------+
|  [ 🎒 裝備 ]   [ 💡 建議 ]   [ 💬 雜項 ]        |
+--------------------------------------------------+
|                                                  |
|  [Alex] 10/12 09:00                       [🗑️] |
|  誰有帶攻頂爐？                                  |
|  |                                               |
|  +-- 2 則回覆 (點擊展開/收合) ----------------+  |
|      |                                           |
|      |-- [Bob]: 我有帶一顆 SOTO 的。             |
|      |                                           |
|      L-- [Alex]: 謝啦！那我不帶了。              |
|                                                  |
|  [  回覆  ]                                      |
|  ----------------------------------------------  |
|                                                  |
|  [Carol] 10/12 12:00                             |
|  公糧分配表出來了嗎？                            |
|  [  回覆  ]                                      |
|                                                  |
+--------------------------------------------------+
|  [ 輸入留言...                       ]  ( 送出 )  | <-- (或使用 FAB 浮動按鈕)
+--------------------------------------------------+
| [ 🏔️ 行程 ]      [ 共編 ]      [ ⚙️ 工具 ]      |
+--------------------------------------------------+
```

## 4. 工具與裝備頁 (Tools & Gear - Tab 3)

**情境**：打包時檢查重量、山上查詢緊急資訊。
**重點**：清楚的重量統計、外部連結按鈕。

```txt
+--------------------------------------------------+
|  工具箱                                          |
+--------------------------------------------------+
|  外部資訊                                        |
|  +--------------------------------------------+  |
|  |  ☁️ 開啟 Windy (向陽山屋)                   |  |
|  +--------------------------------------------+  |
|  |  🌡️ 開啟 中央氣象署 (海端鄉)                |  |
|  +--------------------------------------------+  |
|                                                  |
|  我的裝備清單 (離線)                [ + 新增 ]   |
|  [v] 只顯示未打包                                |
|  ----------------------------------------------  |
|  ▼ 睡眠系統 (1200g)                              |
|  [v] 羽絨睡袋              800g        [Edit]    |
|  [ ] 蛋殼睡墊              400g        [Edit]    |
|                                                  |
|  ▼ 炊具與飲食 (550g)                             |
|  [v] 鈦杯                  150g        [Edit]    |
|  [ ] 瓦斯罐 (小)           400g        [Edit]    |
|                                                  |
+--------------------------------------------------+
|  設定                                            |
|  [ 修改暱稱 ]   [ 強制重整資料 ]                 |
+--------------------------------------------------+
|          總重量: 1750 g (1.75 kg)                | <-- (固定底部)
+--------------------------------------------------+
```
