## 互動規範

| 操作         | 回饋                  |
| :----------- | :-------------------- |
| 點擊行程節點 | 展開 BottomSheet 詳情 |
| 長按列表項   | 顯示刪除/編輯選項     |
| 下拉列表     | 觸發刷新              |
| FAB 點擊     | 新增對話框            |
| 表單提交     | 防連點 + Loading      |
