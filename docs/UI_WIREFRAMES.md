## 4. 行程執行頁 (Itinerary Screen - Tab 1)

**情境**：登山進行中，查看時間軸並進行打卡。
**重點**：高對比度、明確的「預計 vs 實際」狀態、累積距離顯示。

```txt
+--------------------------------------------------+
|  SummitMate                          ➕ ☁️ ⚙️     | <-- 新增按鈕、上傳、設定
+--------------------------------------------------+
|   [ D0 交通 ]   [ D1 上山 ]   [ D2 下山 ]        |
+--------------------------------------------------+
|                                                  |
|  07:00 (預計)           累計 0.0 km              |
|    |                                             |
|    ●  向陽登山口                                  |
|    |  H: 2300m                                   |
|    |  [ ✔️ 實際: 07:05 ]                         | <-- 已打卡
|    |                                             |
+--------------------------------------------------+
|  08:30 (預計)           累計 4.3 km              |
|    |                                             |
|   ◉  向陽山屋 (大休)                             | <-- 目前目標
|    |  H: 2850m                                   |
|    |  [ 📍 點擊查看詳情 ]                        |
|    |                                             |
+--------------------------------------------------+
|  12:00 (預計)           累計 10.2 km             |
|    |                                             |
|    ○  嘉明湖避難山屋                              |
|    |  H: 3380m                                   |
|    |  [ 等待中... ]                              |
|    |                                             |
+--------------------------------------------------+
| [ 🗺️ 行程 ]  [ 💬 協作 ]  [ 🎒 裝備 ]  [ ℹ️ 資訊 ] |
+--------------------------------------------------+
```

## 9. 側邊選單 (App Drawer)

**情境**：從主畫面左側滑出或點擊 hamburger 選單開啟。
**重點**：揪團活動的主要入口點。

```txt
+--------------------------------------------------+
|                                                  |
|  🏔️ [當前行程名稱]                               |
|      3 天行程                                    |
|                                                  |
+--------------------------------------------------+
|                                                  |
|  📋  管理行程                                    |
|      共 5 個行程                            >    |
|  ------------------------------------------------|
|  🥾  揪團活動                               ★★   |  <-- 揪團入口
|      找隊友 / 建立揪團                      >    |
|  ------------------------------------------------|
|                                                  |
|  📅 進行中 / 未來行程                            |
|  +--------------------------------------------+  |
|  |  ○  嘉明湖三天兩夜          07/15-07/17   |  |
|  |  ●  雪山主東單攻 (進行中)    07/20         |  |
|  +--------------------------------------------+  |
|                                                  |
|  📦 已封存 / 結束行程                            |
|  +--------------------------------------------+  |
|  |  ○  玉山群峰五日            06/01-06/05   |  |
|  +--------------------------------------------+  |
|                                                  |
|  ------------------------------------------------|
|  🐻 Alice                    [登出]              |
|                                                  |
+--------------------------------------------------+
```

> [!NOTE]
> 「揪團活動」為揪團功能的**唯一入口**，點擊後進入揪團列表頁。
> 訪客模式下仍可點擊進入，但只能瀏覽列表，無法查看詳情或報名。

**重點**：主要內容區塊限制最大寬度，避免過寬。

```txt
Screen Width > 600px

+-----------------------------------------------------------------------+
|                                                                       |
|         |                                           |                 |
|         |         <-- Max Width 600px -->           |                 |
| Background       +-----------------------+                            |
|  Color  |        | SummitMate         ⚙️ |          |  Background     |
|         |        +-----------------------+          |    Color        |
|         |        | [D0] [D1] [D2]        |          |                 |
|         |        |                       |          |                 |
|         |        | 07:00                 |          |                 |
|         |        |  |                    |          |                 |
|         |        |  O 向陽山屋            |          |                 |
|         |        |                       |          |                 |
|         |        +-----------------------+          |                 |
|         |        | [M] [C] [G] [I]       |          |                 |
|         |        +-----------------------+          |                 |
|         |                                           |                 |
|                                                                       |
+-----------------------------------------------------------------------+
```
