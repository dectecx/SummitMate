## 4. 行程執行頁 (Itinerary Screen - Tab 1)

**情境**：登山進行中，查看時間軸並進行打卡。
**重點**：高對比度、明確的「預計 vs 實際」狀態、累積距離顯示。

```txt
+--------------------------------------------------+
|  SummitMate                          ➕ ☁️ ⚙️     | <-- 新增按鈕、上傳、設定
+--------------------------------------------------+
|   [ D0 交通 ]   [ D1 上山 ]   [ D2 下山 ]        |
+--------------------------------------------------+
|                                                  |
|  07:00 (預計)           累計 0.0 km              |
|    |                                             |
|    ●  向陽登山口                                  |
|    |  H: 2300m                                   |
|    |  [ ✔️ 實際: 07:05 ]                         | <-- 已打卡
|    |                                             |
+--------------------------------------------------+
|  08:30 (預計)           累計 4.3 km              |
|    |                                             |
|   ◉  向陽山屋 (大休)                             | <-- 目前目標
|    |  H: 2850m                                   |
|    |  [ 📍 點擊查看詳情 ]                        |
|    |                                             |
+--------------------------------------------------+
|  12:00 (預計)           累計 10.2 km             |
|    |                                             |
|    ○  嘉明湖避難山屋                              |
|    |  H: 3380m                                   |
|    |  [ 等待中... ]                              |
|    |                                             |
+--------------------------------------------------+
| [ 🗺️ 行程 ]  [ 💬 協作 ]  [ 🎒 裝備 ]  [ ℹ️ 資訊 ] |
+--------------------------------------------------+
```
